# Upload And Update github tutorial
## github
1. edit .gitignore ให้ เรียบร้อย
2. add ทุกไฟล์
```cmd
git add .
```  
3. add commit
```cmd
git commit -m "first commit"
```
4. git push
```cmd
git push -u origin main
```
** ถ้ามีปัญหาการ commit ให้อ่าน Error *
